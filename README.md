## 🇹🇷 Türkçe
### 📝 Hakkında
**Logis**, Linux sistem yöneticileri ve siber güvenlik meraklıları için geliştirilmiş, sistem loglarını otomatik olarak toplayan ve analiz eden bir Python aracıdır. 4 yıllık Linux deneyiminin bir parçası olarak siber güvenlik süreçlerini hızlandırmak için tasarlanmıştır.

### ✨ Özellikler
- 🛡️ **Gelişmiş Analiz:** `/var/log/auth.log`, `syslog` ve `dmesg` dosyalarını tarar.
- 🔍 **Şüpheli Aktivite:** Başarısız giriş denemelerini ve sistem hatalarını tespit eder.
- 📂 **Düzenli Çıktı:** Log verilerini okunabilir bir formatta sunar.

### 🚀 Kurulum ve Kullanım
```bash
# Depoyu klonlayın
git clone [https://github.com/0x4v/Logis.git](https://github.com/0x4v/Logis.git)

# Klasöre gidin
cd Logis

# Aracı çalıştırın
python3 logis.py
